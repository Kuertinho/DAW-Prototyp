## Git & GitHub Workflow

- Remote: `https://github.com/Kuertinho/DAW-Prototyp`
- Default branch: `main`
- After every meaningful change: commit locally with a clean message, then push to `origin/main`
- Commit messages should be concise, imperative-mood subject lines (max ~72 chars), with an optional body explaining *why* not *what*
- Always co-author commits with: `Co-Authored-By: Claude Sonnet 4.6 <noreply@anthropic.com>`
